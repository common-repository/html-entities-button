== Description ==

HTML entities button is a few inserting HTML entities button add to the admin post/page editor(HTML mode).

= Some features: =

* Insert HTML special characters.
* Insert recent post link.
* Insert emoticons.
* Insert wpemoji.
* Insert Dashicons.
* Convert special characters to HTML entities.

* A plug-in installation screen is displayed on the WordPress admin panel.
* It installs it in `wp-content/plugins`.
* The plug-in is made effective.

== Frequently Asked Questions ==

= Emoticon button not found =

Check for [Convert emoticons like :-) and :-P to graphics on display] in Writing Settings.

== Screenshots ==

1. Choose HTML entities.
2. Choose emoticons.
3. Choose recent posts.
4. Convert special characters
5. Choose wpemoji.
6. Entry the font-family.
7. Select the font-family and font-size.
8. Choose Dashicons.
